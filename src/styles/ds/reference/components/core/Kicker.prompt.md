The brand's signature micro-label — mono, UPPERCASE, ultra-wide 0.25em tracking. Sits above almost every Fraunces headline.

```jsx
<Kicker number="05">the white mirror</Kicker>
<Kicker diamond>humanity's requests for startups · v0.1</Kicker>
<Kicker tone="violet">the explanations</Kicker>
```

`number` prints a zero-padded section index in the accent. `diamond` leads with ◆. `tone` recolors the whole label to a terminal signal; the default `muted` keeps grey text with an accent number.
