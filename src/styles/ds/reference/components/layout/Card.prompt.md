A hairline-bordered surface — the workhorse container. Structure comes from the 1px rule, never a shadow.

```jsx
<Card><Kicker number="03">the solutions</Kicker>…</Card>
<Card interactive as="a" href="/companies">What gets built.</Card>
<Card paper>Feature section on the warmer surface.</Card>
```

`interactive` warms the border toward the accent and lifts the surface on hover. `paper` uses the warmer feature surface with a faint sky wash. Default padding 24px.
